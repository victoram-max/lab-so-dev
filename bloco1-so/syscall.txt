# PARTE C — Questões de síntese

# 1. A apresentação mostrou que "instruções de I/O são privilegiadas". 
#    Com base no que o strace revelou, explique com suas palavras como um programa
#    de usuário consegue fazer I/O sem executar instruções privilegiadas diretamente.

Resposta 1:O programa não acessa o hardware diretamente. Ele usa o syscall (tipo write), que é tipo um “pedido” pro kernel fazer isso. 
Quem realmente faz o I/O é o kernel, porque só ele tem permissão pra isso.
